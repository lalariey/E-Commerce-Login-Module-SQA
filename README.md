# E-Commerce Login Module - SQA Assignment

## 1. Project Overview
This project focuses on the Software Quality Assurance (SQA) testing of an E-Commerce Login Module. It includes a comprehensive Test Plan and 10 detailed Test Cases designed to ensure security, usability, and reliability.

## 2. Learning Objectives
* Ability to write clear and concise test cases.
* Mastery of test case management using **Testworthy**.
* Proficiency in classifying test cases (Smoke, Regression, Negative).

## 3. Tools and Technologies
* **Test Management:** Testworthy
* **Version Control:** GitHub
* **Browser:** Google Chrome

## 4. Test Execution Summary
The test suite was executed on Build v1.0.
* **Total Test Cases:** 10
* **Passed:** 9
* **Failed:** 1
* **Pass Rate:** 90%

> **Note:** TC-10 (Session Persistence) failed during execution, identifying a bug in the "Remember Me" functionality.
